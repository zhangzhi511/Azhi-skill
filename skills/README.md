## Add A New Skill

Create a folder with this shape:

```text
skills/
└── skill-name/
    ├── SKILL.md
    ├── scripts/
    ├── examples/
    └── templates/
```

Only `SKILL.md` is required. Add `scripts/`, `examples/`, or `templates/` when the skill needs supporting assets.
